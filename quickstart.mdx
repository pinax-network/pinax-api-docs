## Quickstart

The Pinax Token API lets you access blockchain token information via a GET request. This guide will walk you through how to set up and test an API call.

### Prerequisites
Before you begin, get an API Key by signing up on [The Graph Market](https://thegraph.market/).

<Note> Most Unix-like systems come with cURL preinstalled. For Windows, you may need to install cURL. </Note>

## Using Javascript

Make an API request using **Javascript** by adding the request parameters, and then fetching from the relevant endpoint. For example:
```javascript index.js
const address = "0x2a0c0dbecc7e4d658f48e01e3fa353f44050c208";
const options = {
  method: 'GET',
  headers: {
    'Accept': 'application/json',
    'X-Api-Key': '<...>'
  }
};

fetch(`https://beta.api.pinax.network/balances/evm/${address}`, options)
  .then(response => response.json())
  .then(response => console.log(response))
  .catch(err => console.error(err));
```
Make sure to replace `'X-Api-Key': '<...>'` with your API key, replacing the entire `<...>` with your key. 

For example: `'X-Api-Key': '111aaa222bbb333ccc444ddd555eee666fff777888999000'`

## Using cURL (Command Line)
To make an API request using **cURL**, open your command line and run the following command. Be sure to replace `<...>` with your actual API key:

```curl
curl --request GET \
  --url https://beta.api.pinax.network/v1/balances/evm/0x2a0c0dbecc7e4d658f48e01e3fa353f44050c208 \
  --header 'Accept: application/json' \
  --header 'X-Api-Key: <...>'
```

## Authentication

All API endpoints are authenticated using an API key inserted in the header as `X-Api-Key`.

You can obtain an API key by signing up on the [The Graph Market](https://thegraph.market/).

```json
{
  "headers": {
    "X-Api-Key": "<...>"
  }
}
```

## Troubleshooting

- Debugging:
If the API call fails, try printing out the full response object for additional error details. For example:
```javascript index.js
fetch(`https://beta.api.pinax.network/balances/evm/${address}`, options)
  .then(response => {
    console.log('Status Code:', response.status);
    return response.json();
  })
  .then(data => console.log(data))
  .catch(err => console.error('Error:', err));
```
